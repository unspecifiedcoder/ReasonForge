## ‚ö° The Forging Lifecycle

ReasonForge operates on a sophisticated five-phase lifecycle to ensure maximum logical density and zero hallucination.

1.  **Task Dispatch**: The Subnet Generator creates high-difficulty reasoning challenges across Math, Code, Science, and Ethics.
2.  **Logic Mining**: Miner Nodes (e.g., `0xa3f7`) execute deep, multi-step reasoning chains, providing evidence for every logical leap.
3.  **Protocol Consensus**: Validators analyze the reasoning trace, scoring **Accuracy**, **Completeness**, and **Novelty** to derive a Composite Miner Score (CMS).
4.  **Adversarial Debate**: An AI adversary attempts to find subtle flaws in the logic, forcing the miner to provide a formal rebuttal.
5.  **Intelligence Synthesis**: Final verified logic is compiled into enterprise-grade reports for legal, financial, or scientific deployment.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/unspecifiedcoder/ReasonForge.git

# Navigate to the directory
cd ReasonForge

# Install dependencies
npm install
```

### Development

```bash
# Start the Vite development server
npm run dev
```

### Configuration

To enable **Live Network** mode, provide a Gemini API key in `app.tsx`:
```javascript
const apiKey = "YOUR_GEMINI_API_KEY";
```
*If left empty, the system automatically defaults to **Simulation Mode**.*
